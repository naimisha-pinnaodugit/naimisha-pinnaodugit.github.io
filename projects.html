Misinformation Survey Webportal

A webportal named ElitePro has been designed using Javascript programming language with Nodejs framework. This webportal provides the end user an ability to search amongst the fake news articles by providing a search keyword as input. For the searching operation Elasticcloud search engine API has been used. Information regarding the correctness of the fakenews is provided along with the list of relevant scientific papers that prove the credibility of the fakenews.


Object Counter

Object Counter has been developed which counts the number of objects passing on a conveyor belt. There are 3 systems in this project they are input system, processing system and output system. The input system consists of the sensor section which senses the object passing on the belt .The processing system consists of the incrementation section where the count gets incremented by one when the objects pass one after the other.The output system consists of a 7 segment LED display that displays the number of objects passed
